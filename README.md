# 🚀 TS Newsletter

Aplicação de inscrição em newsletter desenvolvida com **React + TypeScript**, focada em **validação de formulários**, **tipagem segura** e boas práticas de organização de código.

---

## 🧠 Objetivo

Praticar conceitos essenciais de front-end moderno:

- Manipulação de formulários com React
- Validação de dados
- Tipagem forte com TypeScript
- Separação de responsabilidades (SRP)

---

## 🛠️ Tecnologias

- React
- TypeScript
- Tailwind CSS
- Vite

---

## ⚙️ Funcionalidades

- ✅ Validação de campos (nome, email e termos)
- ⚠️ Exibição de erros abaixo dos inputs
- 🔄 Reset do formulário após envio
- 🔐 Tipagem segura com TypeScript
- 🧱 Código organizado e modular

---

## 📂 Estrutura

src/
├── components/
│ └── Form.tsx
├── types/
│ └── User.ts
└── utils/
└── validate.ts

---

## 🚀 Como rodar

```bash
git clone https://github.com/joaoaquino-dev/ts-newsletter.git
cd ts-newsletter
npm install
npm run dev
```
